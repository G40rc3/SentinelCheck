# Sentinel Check Website

This package contains the Sentinel Check website with Ghost Protocol integrated as a 17-module public privacy toolkit.

## Ghost Protocol Scheduler future work

The old Scheduler module is intentionally not part of the public Ghost Protocol navbar at the moment. The current public left navbar should show `17 Intel Feed` as the final module.

Keep this note so the future idea is not forgotten: Scheduler may return later as module 18, but it should be rebuilt as `Broker Follow-Up` or `Opt-Out Follow-Up`, not as a generic maintenance calendar.

The future version should focus on data broker opt-out follow-ups. A static website can support manual tracking, local progress, status labels, copyable follow-up emails, and links back to the Broker Opt-Out module. True automatic resending of opt-out emails would require backend infrastructure, authentication, a database, email sending, scheduled jobs, consent tracking, logs, retention rules, deletion handling, and UK GDPR documentation.

Do not publicly re-add Scheduler until the navbar, module count, page copy, reset behaviour, and privacy wording are all consistent.
